# wix-react-native-print

Printing library

## Installation

```sh
npm install wix-react-native-print
```

## Usage

```js
import WixReactNativePrint from "wix-react-native-print";

// ...

await WixReactNativePrint.print({filePath: "https://static.wix.com/doc.pdf"});
```
